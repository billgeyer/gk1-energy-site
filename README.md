# GK1 Energy Site

Static marketing site for Bill Geyer / GK1 Energy — an independent energy advisor (solar, battery storage, heat pumps, generators). Two hand-coded HTML pages, no build step, no framework.

Deploys manually via Spaceship/cPanel — pushing to GitHub does **not** update the live site. See [`CLAUDE.md`](./CLAUDE.md) for hosting details, site strategy, and the open checklist.
